agentapi-ui とは

agentapi-ui は、agentapi の Web ベース管理インターフェースです。AI エージェントとの対話セッションを管理し、プロファイル設定、Webhook 連携などの機能を提供します。

このドキュメントについて

このドキュメントでは、agentapi-ui の使い方、設定方法、機能の詳細について説明します。
