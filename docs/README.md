# agentapi-ui ドキュメント

agentapi-ui の完全なドキュメントへようこそ。

## agentapi-ui とは

agentapi-ui は、[agentapi](https://github.com/coder/agentapi) の Web ベース管理インターフェースです。AI エージェントとの対話セッションを管理し、プロファイル設定、Webhook 連携などの機能を提供します。

## このドキュメントについて

このドキュメントでは、agentapi-ui の使い方、設定方法、機能の詳細について説明します。

## 主な内容

### はじめに
- **クイックスタート**: 初めて使う方向けのガイド
- **基本概念**: agentapi-ui の基本的な考え方

### Webhook
- **Webhook の設定**: GitHub や外部サービスとの連携
- **Custom Webhook**: JSONPath を使った高度な条件設定
- **トラブルシューティング**: よくある問題と解決方法

### API
- **API リファレンス**: REST API の仕様
- **認証**: API キーと OAuth の設定

### デプロイメント
- **インストール**: 各種環境へのデプロイ方法
- **設定**: 環境変数と設定ファイル
