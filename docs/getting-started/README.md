# クイックスタート

agentapi-ui を使い始めるための基本的な手順を説明します。

## インストール

```bash
# リポジトリをクローン
git clone https://github.com/takutakahashi/agentapi-ui.git
cd agentapi-ui

# 依存関係をインストール
bun install

# 開発サーバーを起動
bun run dev
```

## 基本的な使い方

1. ブラウザで `http://localhost:3000` を開く
2. プロファイルを作成
3. セッションを開始

詳細は[基本概念](concepts.md)を参照してください。
